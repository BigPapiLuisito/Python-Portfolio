# Python-Portfolio
These are the assignments/project that I have created during my year in AP Computer Science Principles

Search 🔍 - This file compares the efficiency of binary search and linear search by demonstrating how their different algorithms impact performance on sorted and unsorted data.

Vacation 🏖️ - This program displays a themed vacation recommendation by selecting an image URL and matching description from stored arrays whenever the user requests one.

Movie App 🎞️ - This program allows users to input specific movie‑related parameters and generates a customized output based on those constraints.

Grocery Stores 🍚 - This program identifies grocery prices and organizes them into separate lists based on item categories and cost.

Dog Breeds 🐶 - This program uses dog breeds as its theme to fulfill the CREATE task by allowing the user to interact with data, apply program‑designed algorithms, and generate meaningful outputs based on their choices.
